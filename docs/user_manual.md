# QR Storage App - User Manual

## Overview
Box Storage App helps users manage physical storage boxes using auto generated box numbers.  
Users can add, view, edit, delete, and search boxes.

---

## Features

1. **Home Screen**
   - Lists all storage boxes
   - Search filter by title, description, or location
   - Tap a box to see details

2. **Add Box**
   - Form fields: Box Number, Title, Description, Location
   - Save button to store new box

3. **Box Detail**
   - Shows box info
   - Options to edit or delete box

---

## How to Use

1. Open the app  
2. Add a new box via the **Add Box** button  
3. Fill out the form and tap **Save**  
4. Return to Home Screen to see the new box  
5. Tap a box to view, edit, or delete  

---

## Notes

- Location is optional
- Box numbers are auto generated
- Search is case-insensitive
